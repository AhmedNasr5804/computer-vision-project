# Paper Figures Directory Description

This folder contains publication-ready figures exported from analysis notebooks.

Figure groups include:

- dataset/data-quality visualizations,
- classical baseline diagnostics,
- CNN and transfer-learning curves,
- confusion matrices and misclassification examples,
- robustness and lightweight-model comparisons,
- deployment benchmark visuals.

These files are imported by `paper/paper.tex` through `\\graphicspath{{figures/}}`.
