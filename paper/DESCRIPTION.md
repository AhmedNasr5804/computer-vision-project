# Paper Directory Description

This directory holds the report manuscript and publication assets.

Key files:

- `paper.tex`: main IEEE paper source.
- `summary.md`: concise numerical summary used for quick reporting.
- `table1_eye.csv`, `table1_lane.csv`: tabular model comparison data.
- `figures/`: final figure images consumed by LaTeX.
- `paper.pdf` and auxiliary files (`.aux`, `.log`, `.out`, `.synctex.gz`): compiled outputs.

The paper references metrics and figures generated from notebook artifacts.
