# Notebooks Directory Description

This directory contains the reproducible experiment pipeline.

- `00_setup_environment.ipynb`: environment/bootstrap notebook.
- `eye/`: full staged eye-state pipeline (01 to 08).
- `lane/`: full staged lane-direction pipeline (01 to 08).
- `09_paper_figures.ipynb`: regenerates publication-ready figures and summary outputs.

Recommended execution order is strictly numeric.
